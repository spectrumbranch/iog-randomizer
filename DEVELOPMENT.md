# Development

## Running tests

See also: https://docs.pytest.org/en/stable/explanation/goodpractices.html

(Do one time) Install pytest for yourself

```pip install -U pytest```

(Do to get venv) Install package in editable mode (optional if you already did it and know what you're doing, just make sure you have a venv then.)

```pip install -e .```

(Repeatable) Run the tests

```pytest```
